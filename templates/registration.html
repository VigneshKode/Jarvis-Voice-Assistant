Personalized Voice Assistant

This Flask application integrates various functionalities, including voice input processing, text generation using OpenAI's GPT-3, database management, and web browsing automation. The application allows users to interact either by typing queries or through voice input.

Upon receiving user input, the application processes it to perform various actions such as fetching the current time or date, opening specific websites like YouTube or Spotify, adding contacts, or generating responses using GPT-3. The conversational history is stored in a SQLite database, allowing users to review past interactions.

Additionally, users can register and authenticate to access personalized features like adding contacts. The application also enables making calls to contacts using the Android Debug Bridge (ADB) utility.

Overall, this application provides a versatile platform for interacting with AI-generated responses, managing contacts, and performing web-related tasks, all within a Flask-based web framework. It demonstrates the integration of multiple technologies to create a cohesive user experience.
